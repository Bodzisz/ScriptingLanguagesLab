Query options:

- Set total on/off      -> sets total option to on or off
- Show (+ conditions)   -> makes request to show data with given conditions
- exit                  -> exits program

Available conditions:
- in {country}
- in {month}
- in {continent}
- from {month} {day}
- till {month} {day}

You can also specify if you want to see cases of deaths (cases is default):
- Show cases/deaths (+ conditions)

Conditions are not neccessary, you can just run:
- Show
